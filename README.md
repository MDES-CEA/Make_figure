# Générateur de figures DRX / Raman

Outil web statique de génération de diffractogrammes empilés (DRX) et spectres Raman,
avec fiches de référence PDF (.dif Bruker EVA) et personnalisation complète du rendu.

Les données restent locales au navigateur : aucun fichier n'est envoyé sur un serveur.

## Développement local

    npm install
    npm run dev

## Build

    npm run build

Application React + Vite + Tailwind CSS v4.
